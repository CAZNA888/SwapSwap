# Audio

Звуковые файлы для игры. Формат: MP3

## Необходимые файлы:

- **CardDeal.mp3** - Звук раздачи карты (воспроизводится при каждой карточке)
- **CardFlip.mp3** - Звук переворота карты (воспроизводится при каждом перевороте)
- **Swipe.mp3** - Звук свайпа (воспроизводится при перемещении карточки)
- **LevelComplete.mp3** - Звук победы (воспроизводится при завершении уровня)

## Настройки импорта (рекомендуемые):

- **Compression Format**: Compressed
- **Quality**: 70% (для оптимизации размера)
- **Load Type**: Compressed In Memory
- **Sample Rate**: 22050 Hz (для уменьшения размера)

## Использование:

Звуки автоматически загружаются в AudioManager через инспектор Unity.
